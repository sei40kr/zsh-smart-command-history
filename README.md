# zsh-smart-command-history

Prevent storing failed commands in the history list.

## Install

### Install with zinit (recommended)

```sh
zinit light sei40kr/zsh-smart-command-history
```

### Install with zplug

```sh
zplug sei40kr/zsh-smart-command-history
```

## Supported Options

* `HIST_IGNORE_SPACE`
* `HIST_REDUCE_BLANKS`
