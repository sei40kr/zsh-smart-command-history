# zsh-smart-history

Keep from saving failed commands to the history.

## Install

### Install with zinit (recommended)

```sh
zinit light sei40kr/zsh-smart-history
```

### Install with zplug

```sh
zplug sei40kr/zsh-smart-history
```

## Supported Options

* `HIST_IGNORE_SPACE`
* `HIST_REDUCE_BLANKS`
